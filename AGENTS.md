# AGENTS.md

## Project overview

A web tool that displays route tracks from GPX files on a map.

Tech stack:

- Node.js
- TypeScript

## Build & test commands

- Install dependencies: `npm i`
- Build: `npm run build`
- Debug run: `npm run dev`
- Lint: `npm run lint`
- Lint auto-fix: `npm run lint:fix`
- Format: `npm run format`
- Check formatting diff: `npm run format:check`
- Test: `npm test`

## Coding conventions

- File names: kebab-case (`example-file-name.ts`)
- Write comments, documentation comments (JSDoc) and git commit messages in English
- Write documentation (README.md, etc.) in English

## Workflow

- Create a topic branch for each unit of work and merge it into `master` via a pull request
